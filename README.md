# siddhartha-biswas-openapi

A simple project that fetches weather data from an API.
The project contains an `index.html` file and two folders: `css` and `js`.

## Project Structure

```
siddhartha-biswas-openapi/
│
├── index.html
├── css/
│   └── index.css
└── js/
    └── index.js
```

## How to Run the Project

1. **Download or clone the repository**

2. **Run the project**

   * Simply open the `index.html` file in your web browser.


## Features

* Fetches data from an Weather API and have multiple Fetches
* Simple HTML, CSS, and JavaScript structure
* Easy to modify and extend

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
